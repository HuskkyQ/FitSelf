# 健身运动高频缩写对照表

## 一、强度评估类
| 缩写 | 英文全称 | 中文释义 |
| ---- | ---- | ---- |
| RPE | Rate of Perceived Exertion | 主观用力程度分级 |
| RIR | Reps In Reserve | 剩余次数 |
| MET | Metabolic Equivalent | 代谢当量 |
| HR | Heart Rate | 心率 |
| HRmax | Maximum Heart Rate | 最大心率 |
| BPM | Beats Per Minute | 每分钟心跳数 |

## 二、组次与动作节奏
| 缩写 | 英文全称 | 中文释义 |
| ---- | ---- | ---- |
| Rep/R | Repetition | 单次动作/次数 |
| Set/S | Set | 组 |
| RM | Repetition Maximum | 最大重复次数 |
| Rst | Rest | 组间休息 |
| TUT | Time Under Tension | 肌肉受力时长/动作节奏 |
| ECC | Eccentric | 离心阶段 |
| CON | Concentric | 向心阶段 |

## 三、训练模式
| 缩写 | 英文全称 | 中文释义 |
| ---- | ---- | ---- |
| AMRAP | As Many Rounds/Reps As Possible | 限时尽力做多（次数/组数） |
| EMOM | Every Minute On the Minute | 每分钟定时训练 |
| HIIT | High-Intensity Interval Training | 高强度间歇训练 |
| LISS | Low-Intensity Steady State | 低强度稳态有氧 |
| MICT | Moderate-Intensity Continuous Training | 中等强度持续有氧 |
| Superset | Superset | 超级组 |
| Drop Set | Drop Set | 递减组 |
| GI | Giant Set | 巨人组 |
| PPL | Push/Pull/Legs | 推拉腿（训练分化） |
| FBW | Full Body Workout | 全身训练 |
